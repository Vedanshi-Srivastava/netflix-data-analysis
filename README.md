# 🎬 Netflix Data Analysis

This project analyzes Netflix's content dataset to uncover trends in content production, genres, ratings, and growth over time using Python.

---

## 🚀 Key Highlights

- Analyzed Movies vs TV Shows distribution  
- Studied content growth trends over the years  
- Identified top countries producing content  
- Explored popular genres and ratings  
- Performed data cleaning and preprocessing  

---

## 🛠 Tech Stack

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 📊 Key Insights

- Netflix has more Movies than TV Shows  
- Content growth increased rapidly after 2015  
- USA is the leading content producer  
- Drama & International genres dominate  
- Most content is targeted toward mature audiences
